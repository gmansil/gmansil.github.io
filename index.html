HTML

HTML (Hypertext Markup Language) is a markup language used for creating web pages and applications. It allows developers to structure content on a web page by using various elements such as headings, paragraphs, lists, images, and links. HTML provides the basic structure of a web page and determines how the content should be displayed in a browser. It consists of tags enclosed in angle brackets that indicate what type of content is enclosed within them.

If you're following along through a text editor, start by creating an HTML file: index.html. View your file with a live server to see our web page change as we progress!

Create a DOCTYPE(if you are using VS Code you can simply type "!" and hit tab. If not google "HTML doctype" and copy and paste. This is already in the index.html file for people using replit.

Edit the title of your site by editing the title tag in the head of the file.

Now in the scope of the body tag, create a heading using the <h1> which is known as the header tag and put your name in the contents.

<h1>Your name</h1>

Create a paragraph using the <p> tag tag and add a few defining words about yourself.

<p>Student and Future Occupation</p>

Create another paragraph and add a sentence about yourself

<p>I am a student at Swarthmore College and an aspiring occupation.<p>

Now create a <hr> to provide more structure to the page. Notice that the horizontal rule tag does not have a closing tag.

Let's now create another header but using the <h3> tag instead this time and put "Education" in between the tags.

We will now create a list using the <ul> tag. To create a list item, enter the scope of your <ul> tag and create a <li>. For each item of your list create a new <li>.

<ul>

     <li>Highschool at SECA</li>

     <li>Currently enrolled at Swarthmore<li>

</ul>

Create another subsection but this time for Work Experience. Each time we create a subsection, use a horizontal rule to separate the content of your web page.

CSS

CSS (Cascading Style Sheets) is a style sheet language used to describe the presentation of a document written in HTML. It provides a way for developers to define the visual appearance of elements on a web page, including layout, colors, fonts, and sizes. By separating the presentation of a web page from its content, CSS allows developers to make changes to the layout and appearance of a web page without changing the underlying HTML code. CSS is written using a set of rules that define the style of an element, which are then applied to the appropriate HTML elements. The latest version of CSS is CSS3, which has introduced many new features such as animations, gradients, and transitions.

Their are many ways to add styling to a website. Some websites have separate files which have styling instructions that can be applied to tags. We will use in-line tagging for the sake of simplicity. This is an example of in-line styling:

<hr size="3">

The styling for this horizontal rule is pretty straightforward. An important note is that the styling for this tag will not apply to any other horizontal tags.

When using in-line styling, we can omit the style attribute if the styling isn't CSS styling. Changing the size of our horizontal rule isn't specific to CSS.

Add the following in-line styling to your <hr style=> tag

border-style: none; border-top-style: dotted; border-color: gray;        border-width: 5px; width: 5%;

If you want to change the background color of your website, add in-line styling to your <body color:40514E>

Because of how the HTML has been structured, the body of the web page lies within the body tag limiting our styling options. A good way of thinking about modern web development is as a collection of rectangles which are created by the <div> tag. Web devlopment becomes complex when choosing the positioning of divs and ensuring that the flow of web page isn't awkward.

Adding Images

Any modern day website uses images so we will use them too. Lets start by creating a folder called images. Add the image of your choice, can be a headshot or any random picture that you like, to your folder(you can drag and drop).

Now back in our HTML file, we will create a image tag right above the heading with our name <img src="images/placeholder"> and replace the placeholder with the file path of the image. If you need help with this let me know. To add an image to a repl, upload the image from your computer. Depending on how your image is saved, you may need to use a file converter (google an image converter).

Depending on the image you have used, the size of the image can differ greatly based on the image's original dimensions or resolution. We can use in-line styling to limit the height and width of our image. <img src="images/placeholder" style="border-radius: 100%">. This makes our image circular which fits if you decided to use a headshot. Other ways of styling images is by editing their height or width.

Feel free to google styling for any of the tags used so far. You can also make your image smaller or larger depending on what is necessary for your webpage.

If you would like to add more images of yourself playing sports or pets or anything of interest feel free to rinse and repeat. The placement of your images will drastically change the format of your webpage.
